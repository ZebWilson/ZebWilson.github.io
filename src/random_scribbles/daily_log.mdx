steps: Basically you have source and relay (our C6).

2. Make source and relay use same channel (36)
3. Make source and relay use same encryption pattern (WPA2-PSK)
4. Disable EasyMesh on relay (C6).
5. Set LAN IP of relay such that it's inside the x.x.x part of the ip.
   Eg: If source is 192.168.1.1 and relay is at 192.168.0.1 change relay into 192.168.1.120 (any value from 2-254
